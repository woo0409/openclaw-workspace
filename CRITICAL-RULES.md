# 🚨 关键操作规则 - 绝对禁止违反

## 📊 数据库删除规则
**规则: 任何涉及数据库删除的操作，必须先获得主人明确确认！**

### ❌ 禁止的操作（未确认）
- 删除MySQL容器
- 删除数据库数据卷
- 清空数据库表
- 执行 DELETE/ DROP/ TRUNCATE 语句
- 删除包含数据库的Docker镜像

### ✅ 正确操作流程
1. 检查是否有数据卷依赖
2. 备份数据（如果需要）
3. **询问主人确认**
4. 得到明确确认后再执行

### 📝 验证方法
```bash
# 检查数据卷
docker volume ls

# 检查数据卷中的数据
ls -la /var/lib/docker/volumes/<volume_name>/_data/

# 导出备份
mysqldump -u root -p database_name > backup.sql
```

## 🎯 原则
**数据无价，删除必问！**
